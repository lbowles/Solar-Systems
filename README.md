# Solar Systems

This is the code repository for Solar Systems, a fully on-chain, procedurally generated, animated solar systems NFT collection.

## Stack

### Backend

- Hardhat
- Typescript
- Typechain

### Frontend

- React
- wagmi
- RainbowKit

## Usage

### Cloning

```
git clone <repo name>
```

### `backend`

```
yarn
```

```
yarn start
```

### `frontend`

```
yarn
```

```
yarn start
```
